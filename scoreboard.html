Scoreboard Operation Guide

Scoring: Use +1 buttons for Team A (yellow) or Team B (orange)
Timer: START to begin, HOLD to pause, RESET to restore original time
Fouls: Track team fouls with +1 buttons under each team
Timeouts: Add timeouts for each team using the timeout +1 buttons
Possession: Toggle possession arrow between teams
Error Correction: Use ERROR button to undo last action
